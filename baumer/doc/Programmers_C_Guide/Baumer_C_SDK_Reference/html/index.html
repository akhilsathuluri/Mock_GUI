Welcome to the Baumer GAPI SDK for C

Welcome to the new Baumer GAPI SDK for C. Baumer GAPI is the proven solution to to integrate GenICam Cameras stable and highly performant. The Baumer GAPI SDK also supports programming in C++ and C# and is available for your choice of architecture; Windows x86 and x86-64, Linux x86-64, ARMhf and AArch64.

You can download the Baumer GAPI SDK for your preferred architecture from the Baumer Website.

Introduction to the Baumer GAPI SDK

Since more than 15 years, Baumer provides the Baumer GAPI SDK to help you integrate GenICam camera devices. The Baumer GAPI is widely used by numerous camera integrators in many industries. As camera hardware and computing devices becomes more powerful, more and more image recognition tasks are enabled and the usage of cameras in industrial application has skyrocketed. With the Baumer GAPI we offer full flexibility and full control to the expert users. The Baumer GAPI SDK targets single and multi-camera systems with highest performance requirements.


The Baumer GAPI SDK for C is based on the GenICam reference parser. It features

  • Producers for U3V, GigE and 10GigE and support for GenTL 1.5
  • Compatibility with GenICam cameras supporting SFNC up to version 2.3
  • Node (feature), stream, chunk and event-handling
  • The GUI Tool Camera Explorer to evaluate and provision cameras and as a monitoring solution
  • The IP-Config-Tool for the network setup of GigE cameras
  • Numerous examples to help getting started
  • This documentation

Feel free to start browsing the API documentation now or read further to get more information about the underlying concepts or more help on how to get the API working on your system.
